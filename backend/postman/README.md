# Postman Collection for OurRide API

## Quick Start

1. **Import Collection:**
   - Open Postman
   - Click "Import"
   - Select `OurRide_API_Collection.json`

2. **Import Environment:**
   - Click "Environments"
   - Import `OurRide_Environment.json`
   - Select "OurRide - Local Development"

3. **Start Backend:**
   ```bash
   mvn spring-boot:run
   ```

4. **Test Endpoints:**
   - Start with "Send OTP"
   - Check console for OTP
   - Verify OTP to get tokens
   - Tokens auto-saved for other requests

## Files

- `OurRide_API_Collection.json` - Complete API collection
- `OurRide_Environment.json` - Environment variables
- `API_TESTING_GUIDE.md` - Detailed testing guide
